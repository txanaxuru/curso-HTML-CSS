Principais formatações

Negrito/Destaque

Nesta frase, temos um termo em negrito usando a tag B-(não semântica/apenas sentido de forma)

Nesta frase, temos um termo em destaque usando a tag STRONG-(semântica/com sentido real de dar destaque a determinada parte do texto.)

Itálico/ênfase

Nesta frase, temos um termo em itálico usando a tag I-(sem semântica)

Nesta frase, temos um termo com ênfase usando a tag EM-(semântica)

Texto marcado

Para criar uma marcação de texto utilizamos a tag MARK(sem semântica)

Para modificar a cor individualmente usamos o parâmetro css style-background color e selecionamos a cor ou também padronizar colocando o style no cabeçario ex:rosa

Texto grande e pequeno

Nesta frase estamos criando um texto grande(tag BIG-ja obsoleta não a sentido de usa-lá pois não há função semântica e se for necessário modificar a forma de alguma letra o melhor caminho é através do css) e um texto pequeno(tag SMALL-ainda válida por ter sentido semântico, ou seja, ainda tem propósito além de apenas sua forma e pode ser usada por exemplo em contratos nas ditas letras menores...)

Texto deletado

podemos marcar um texto como excluído para identificar que ele deve ser lido, mas não considerado, usando a tag DEL(semântica)

Texto iserido

podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag INS(semântica)

ou também podemos inserir com a tag U(que não possuí semântica)

Texto sobrescrito

Utiliza-se a tag SUP(em cima) para inserir coisa do tipo x7+7=?

Texto subscrito

Utiliza-se a tag SUB(em baixo) para inserir coisas do tipo H2O
